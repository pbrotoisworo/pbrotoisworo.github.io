Tweet2Map

Open-source traffic accident database for Metro Manila


LATEST DATASET VERSION: December 20, 2020
CURRENT DATASET SIZE: 17,000~ INCIDENTS

Download the dataset for free on Kaggle
Download the source code on Github


Tweet2Map is a semi-automated process of getting traffic incident Tweets from the Metro Manila Development Authority (MMDA) and then parsing the information into a usable database. I will be building a database over time and sharing the output with everyone.
The interactive map below shows the latest 1,000 incidents in the database. Check this page for updates!

Disclaimer: There may be some errors and some gaps related to the acquisition and parsing of the Twitter data but most of it should be accurate. Read more about MMDA Tweet2Map code on this blog post.
